Site Name

PhysiCore is an educational site dedicated to helping beginners understand physics principles in a visual and practical way.

Site Purpose

The site provides beginner-friendly resources to learn physics, especially mechanics. It uses interactive JavaScript tools (like velocity and force calculators), visual aids, and simple explanations to make learning engaging and practical for students and educators.

Scenarios

  • What is the difference between velocity and acceleration?
  • Is there a simple way to calculate force online?

Color Schema

The color palette includes:

  • Oxford Blue (#002147): used for headers and background accents.
  • Ivory (#FFFFF0): used for the background and content sections.
  • Burgundy(#800020): Used for accents like call-to-action buttons or important links.
  • Gold (#DAA520): Used sparingly for decorative or highlight purposes.

Typography

  • EB Garamond: Used for body text.
  • Heading: Used for headings.
